# 🚀 بوت أتمتة متجر ترند اليمن (Trend Store Yemen Bot)

نظام ذكي يعتمد على الذكاء الاصطناعي لتحليل صور المنتجات وتوليد كلمات مفتاحية للبحث، مع مزامنة كاملة مع Google Sheets.

## 🛠️ التحديثات الأخيرة (نسخة Enterprise)
تم تحديث النظام ليعمل بكفاءة 24/7 على منصة **Railway** مع الميزات التالية:
* **Flask Integration:** نظام "Health Check" لإبقاء السيرفر مستيقظاً وتجنب وضع النوم في Railway.
* **Smart Orchestrator:** محرك فحص دوري (كل 30-60 ثانية) يعالج المهام تلقائياً.
* **Case-Insensitive Mapping:** نظام فحص مرن لحالة المهام (Pending/pending).
* **Robust Sheet Mapping:** ربط مباشر بالأعمدة (B, C, D) لضمان دقة الكتابة.

## 🏗️ هيكلية النظام (Architecture)
1. **Google Sheets:** قاعدة البيانات وواجهة التحكم.
2. **Railway:** استضافة الكود وضمان استمرارية التشغيل.
3. **Gemini AI:** تحليل الصور واستخراج الكلمات المفتاحية.



## 📊 هيكل جدول البيانات (Google Sheets)
يجب أن يحتوي الجدول على الأعمدة التالية بالترتيب:
* **العمود A:** اسم المنتج أو المعرف.
* **العمود B:** رابط الصورة (ImageURL).
* **العمود C:** حالة المعالجة (Status) - اكتب `Pending` للبدء.
* **العمود D:** النتائج (Keywords).

## 🚀 الرفع والتشغيل
عند رفع الكود، سيبدأ `main.py` بتشغيل سيرفر Flask ومحرك البوت في آن واحد.
